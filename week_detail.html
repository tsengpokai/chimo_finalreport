Week 2: 震測理論與推導

臨界折射波走時與 Crossover Distance 推導

根據 Snell's Law 與幾何路徑，我們推導了雙層介質下的折射波走時公式與直達波/折射波的交會距離 ($X_{cr}$)。

1. 走時公式 (Travel Time)

折射波總時間 \( T_t \) 由下降段、水平段、上升段組成：

$$ T_t = \frac{x}{V_2} + \frac{2h \cos \theta_c}{V_1} $$

其中 \( \theta_c \) 為臨界角，滿足 \( \sin \theta_c = \frac{V_1}{V_2} \)。

2. Crossover Distance (\(X_{cr}\))

當直達波時間等於折射波時間時的距離：

$$ \frac{X_{cr}}{V_1} = \frac{X_{cr}}{V_2} + 2h \frac{\sqrt{V_2^2 - V_1^2}}{V_1 V_2} $$

經過整理可得：

$$ X_{cr} = 2h \sqrt{\frac{V_2 + V_1}{V_2 - V_1}} $$

Week 5: ObsPy 地震波處理

Python 抓取 IRIS 地震資料

使用 ObsPy 套件連接 FDSN Client，下載並繪製 2025/10/07 的地震波形。

from obspy import UTCDateTime
from obspy.clients.fdsn import Client
import matplotlib.pyplot as plt

client = Client("IRIS")
starttime = UTCDateTime("2025-10-07T23:52:12")
duration = 120 
endtime = starttime + duration

# 下載資料
st = client.get_waveforms("TW", "NACB", "", "BHZ", starttime, endtime)

# 繪圖
fig = plt.figure(figsize=(10, 4))
st.plot(outfile="waveform.png", fig=fig)
